# Проект

⚠️ **Важно: в текущей версии проекта отсутствует автоматическая установка зависимостей.**

Запуск проекта вручную может вызвать значительные трудности, так как:
- Необходимо самостоятельно установить все требуемые библиотеки.
- Отсутствует скрипт для настройки окружения

Рекомендуется дождаться появления полноценной системы установки или использовать изолированное виртуальное окружение и устанавливать зависимости вручную, основываясь на импортах в коде.

Работа над автоматизацией развёртывания находится в планах.
